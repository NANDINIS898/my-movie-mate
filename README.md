# 🎬 Movie Mate

**Movie Mate** is a modern movie recommendation web application powered by the TMDB API. It offers personalized movie suggestions based on genres and moods. Users can sign up, log in, explore trending titles, save favorites, and create mood-based playlists.

---

## 🌟 Features

- 🔐 **User Authentication** – Sign Up and Login securely
- 🎞️ **Genre-Based Recommendations** – Discover movies tailored to your favorite genres
- 🎭 **Mood Playlists** – create your mood playlists- downloadable & shareable
- ❤️ **Favorites** – Save your favorite movies for quick access
- 🔎 **Search** – Find any movie using the TMDB database
- 📱 **Responsive UI** – Works seamlessly across all devices

---

## 🛠 Tech Stack

- **Frontend**: React.js, CSS
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL
- **Authentication**: JWT (JSON Web Tokens)
- **API**: TMDB (The Movie Database)
- **Deployment**:
  - Frontend: Netlify
  - Backend: Render 

---

## 🔗 TMDB API

This app uses [The Movie Database (TMDB)](https://www.themoviedb.org/) API to fetch:
- Movie titles, posters, overviews, ratings
- Genre lists
- Search results
- Trending/popular movies

> ⚠️ You need a TMDB API key. Get it from: [TMDB Developer Page](https://developers.themoviedb.org/)
